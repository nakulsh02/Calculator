# Futuristic Calculator

A **futuristic, very modern, and premium calculator** with a sleek user interface and smooth interactions. This project is fully functional and designed with a clean, next-generation look.

## Features

- Modern and **futuristic UI design**
- Smooth button animations and transitions
- Accurate calculations
- Fully **responsive** (mobile, tablet, desktop)
- Easy to use and lightweight
- Runs directly by opening `index.html`

## About

This calculator is **entirely created by me**, including design and functionality.  
All code is original and written from scratch, focusing on both **performance and premium visual experience**.

## Usage

1. Download or clone the repository  
2. Open `index.html` in any browser  
3. Start calculating instantly
